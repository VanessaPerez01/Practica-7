¿Que es Front-End?

Es la parte del desarrollo web que se dedica a la parte frontal de un sitio web, es decir del diseño de un sitio web, yendo desde la estructura del sitio hasta los estilos como colores, fondos, tamaños hasta llegar a las animaciones y efectos.

Fundamentos de Front-End

  • HTML: el organizador

    • • HTML es la forma en que se organizan todos los sitios en la web, por lo que no puede vivir sin el desarrollo de front-end.
    • HTML5 es la última especificación HTML.
  • CSS: el estilista
    • CSS es la forma en que los desarrolladores agregan estilo y efectos a un sitio web. Los estilos se pueden agregar globalmente y luego superponerse sin cambiar ese estilo fundamental que se aplica a todo un sitio.
    • CSS siempre está evolucionando.
    • Muchos desarrolladores de aplicaciones para usuario utilizan un preprocesador de CSS para hacer que escribir muchas líneas de CSS sea aún más rápido.
    • Los marcos de front-end CSS como Foundation o Bootstrap pueden ayudarlo a crear sitios web pulidos en un instante.
